Siteswap Diagram Generator

This is a super simple utility for generating vector files (SVG format) to represent siteswaps. At present, no validation of the siteswap is performed. Only vanilla asynchronous siteswaps with max throwheight 9 are currently supported. If you want a better tool for siteswap visualization, visit juggling.graphics.
